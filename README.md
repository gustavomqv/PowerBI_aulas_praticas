# PowerBI_aulas_praticas
Repositório com Painéis de Power BI de aulas práticas

Neste repositório há um arquivo .pbix com todos os painéis criados nas aulas práticas de Ferramentas de Business Intelligence;

As bases usadas foram fornecidas pelo professor.

Usou-se o power querry para o tratamento e transformação dos dados. As medidas usadas nos painéis foram criadas através do uso de código DAX.
Todos os painéis e indicadores foram criados em cima da base de dados e também de extração pela internet (como por exemplo os estados e bandeiras dos estados). 

E por fim, agluns prints de aglumas páginas do relatório criado.
